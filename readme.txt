Easily add a 24/7 Bangla Radio to your website.

== Description ==

- Easily add a 24/7 Bangla Radio to your website.

- You Can Use/Put This Radio Any Post/Page of Your Website by Using Sortcode.

- After Install A New Widget will Added Named 'Radio Abirvab', You can use this widget to your website sidebar or any widget area.

- Just Use This Sortcode to Display The Radio Player  [radioabirvab]

- We use html5 player so works on cross platform (PC & Mobile)


- Here is Live Demo: http://radio.studioarrival.com

== Installation ==

- Install from WordPress admin panel >> Plugins >> Add new plugin >> Search for `Bangla Radio Abirvab' plugin. After you install, activate the plug-in. 

- After Successfully Install & Active This Plugin You Need to Use Sortcode for display the player. use [radioabirvab] sort code for display the player anywhere in your website. (E,g: Post/Page or Widget Area)

- Or just drag and drop your from widget panel
